D3.js Slider Examples

http://github.com/turban/d3.slider

Default slider

d3.slider()

Slider with start value

d3.slider().value(25)

Slider with slide event: 0

d3.slider().on("drag", function(evt, value) {
  d3.select('#slider3text').text(value);
})

Slider with default axis

d3.slider().axis(true)

Slider with custom axis

d3.slider().axis(d3.svg.axis().orient("top").ticks(4))

Slider with min, max and step values

d3.slider().axis(true).domain([2000, 2100]).step(5)

Vertical slider

d3.slider().value(50).orientation("vertical")

Slider range with slide event: 0

d3.slider().domain([10,90]).on("drag", function(evt, value) {
  d3.select('#slider8text').text(value);
})
